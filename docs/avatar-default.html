Sutando default avatar — 5 states

Inline SVG, per-state CSS keyframes. No external assets. Designed for general users without a custom avatar; custom-identity sutandos keep their PNG and get the same state classes applied to an aura layer around the image.

idle

indigo violet · breathe · faint twinkle

listening

violet · wave pulse · mic ready

speaking

green · visor scroll · arms reach out

working

blue · orbiting dots · tool in flight

seeing

amber · visor scan · looking at screen

Colors: indigo #7c83ff (idle), violet #a78bfa (listening), green #4ecca3 (speaking), blue #60a5fa (working), amber #fbbf24 (seeing). Switching state is a single class toggle on the parent cell (.s-idle → .s-working etc.).
